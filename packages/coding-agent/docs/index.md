# Codeify CLI Documentation

Codeify is the terminal coding agent for [codeify.cc](https://codeify.cc). It runs in the current repository, uses read/write/edit/search/shell tools, stores sessions locally, and sends model requests through the Codeify OpenAI Responses API provider.
